Documentación de EFI

📖 Descripción

Nombre: EFI

Objetivo: Edición rápida de imágenes eliminando fondos automáticamente con IA.

Tecnologías: Python, OpenCV, PIL, rembg, Tkinter

⚙️ Instalación

  • Windows, Linux o macOS
  • RAM: 500MB mínimo, 1GB recomendado
  • GPU: Opcional
git clone https://github.com/jesuspinedaof/efi.git
cd efi
pip install -r requirements.txt

✨ Funcionalidades

  • Carga imágenes desde tu sistema
  • Elimina fondos con IA automáticamente
  • Interfaz gráfica sencilla de usar
